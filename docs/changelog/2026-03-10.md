## [2026-03-10] Blog System

Реализована полноценная блог-система с evidence-based psychology контентом для органического трафика и SEO.

**Основные фичи:**
- Blog landing page с animated tag cloud (интерактивное облако тегов)
- Динамический роутинг статей через Next.js `/blog/[slug]`
- Rich content blocks (intro, stat banners, tables, sources, CTA)
- SEO metadata (OpenGraph, Twitter cards, keywords)
- Фильтрация статей по тегам
- Responsive дизайн в стиле Confide (тёплый, профессиональный)

**Первая статья:**
"Understanding Anxiety: 7 Signs You Might Not Recognize"
- 8 min read
- Evidence-based content с citations (WHO, Lancet, DSM-5-TR)
- Статистические данные в визуальных банерах
- Таблицы с prevalence & treatment efficacy
- CTA к регистрации

**Навигация:**
- Ссылка "Blog" добавлена в desktop navigation на лендинге
- Ссылка "Blog" добавлена в footer на всех публичных страницах (landing, contact, support, refund, privacy, terms)

**Файлы:**
- `lib/blog/articles.ts` — база статей с типизацией
- `lib/blog/tags.ts` — теги с весами для облака
- `components/blog/AnimatedTagCloud.tsx` — интерактивное облако тегов
- `components/blog/ArticleCard.tsx` — карточка статьи с hover анимациями
- `components/blog/ArticleView.tsx` — рендеринг rich content blocks
- `app/(marketing)/blog/page.tsx` — главная страница блога
- `app/(marketing)/blog/[slug]/page.tsx` — динамические страницы статей с SEO
- `app/(marketing)/layout.tsx` — layout для marketing секции

**Результат:**
- ✅ Блог доступен на `/blog`
- ✅ SEO-оптимизированные страницы статей
- ✅ Навигация интегрирована в весь сайт
- ✅ Готово к масштабированию (легко добавлять новые статьи)

## [2026-03-10] SEO Optimization

Добавлены файлы для поисковой оптимизации: динамический sitemap, robots.txt, и JSON-LD structured data для статей блога.

**Файлы:**
- `app/sitemap.ts` — динамический sitemap.xml со всеми публичными страницами и статьями блога
- `app/robots.ts` — robots.txt с правилами индексации (allow public, disallow dashboard/api/auth)
- `app/(marketing)/blog/[slug]/page.tsx` — добавлен JSON-LD structured data (schema.org Article)

**Результат:**
- ✅ `/sitemap.xml` доступен для поисковых ботов (Google, Bing)
- ✅ `/robots.txt` настроен для правильной индексации
- ✅ Статьи блога имеют structured data для rich snippets в поисковой выдаче
- ✅ SEO-готовность для органического трафика
