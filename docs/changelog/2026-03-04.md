# Changelog — 4 марта 2026

## RAG Optimization — Query Expansion ✅

**Проблема:** Средний retrieval score 0.46, 62% FAIL.

**Решение:** Query Expansion через GPT-4o-mini — расширяет короткие запросы в 50-100 слов domain keywords.

**Результат:** Average score 0.46 → 0.64 (+38%). 0% FAIL. Стоимость ~$0.00004/запрос.

**Файлы:** lib/pinecone/retrieval.ts (expandQuery function)

**Результат:** Average score 0.64 → 0.69 (+10.3%). 9/21 PASS (43%). Cumulative: +64%.

**Файлы:** lib/pinecone/reranker.ts, scripts/ingest-knowledge.ts (chunk filtering)

**Best scores:** Anxiety 0.80, Trauma 0.80, Mens 0.80, Cross-agent 0.80.
