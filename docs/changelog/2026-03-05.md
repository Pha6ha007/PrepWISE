## PDF Diary Auto-Generation (Vercel Cron Job) ✅

Автоматическая генерация дневников 1 числа каждого месяца через Vercel Cron Job.

**Cron Schedule:** "0 6 1 * *" (6:00 UTC на 1 число каждого месяца)
**Endpoint:** GET /api/cron/generate-diaries
**Security:** CRON_SECRET auth header (Bearer token защита от внешних вызовов)
**Logic:** Генерирует дневники для предыдущего месяца для всех пользователей с сессиями

**Service Layer Refactoring:**
- lib/diary/service.ts — переиспользуемая функция generateDiaryForUser(userId, month, year)
- app/api/diary/generate/route.ts — обновлен для использования service
- app/api/cron/generate-diaries/route.ts — cron endpoint с batch обработкой

**Process:**
1. Находит всех пользователей с сессиями в предыдущем месяце
2. Запускает Promise.allSettled для параллельной генерации
3. Логирует результаты: generated / skipped / errors
4. Возвращает статистику: totalUsers, generated, skipped, errors, errorDetails

**Configuration:**
- vercel.json — cron definition
- .env.example — CRON_SECRET added (generate: openssl rand -base64 32)

**Идемпотентность:** Если дневник уже существует (status: ready), пропускает генерацию.
**Error handling:** Если генерация упала для одного пользователя, продолжает для остальных.

**Файлы:** vercel.json, lib/diary/service.ts, app/api/cron/generate-diaries/route.ts, app/api/diary/generate/route.ts, .env.example
