# Changelog — 24 марта 2026

## Миграция биллинга: Paddle → Dodo Payments

Полная миграция платёжной системы с Paddle на Dodo Payments (Merchant of Record).

**Что изменилось:**
- Paddle SDK удалён, установлены `dodopayments` + `@dodopayments/nextjs`
- Checkout теперь redirect-based (hosted checkout Dodo), не overlay
- Webhook верифицирует HMAC-SHA256 подпись, обрабатывает: `subscription.active`, `plan_changed`, `renewed`, `cancelled`, `expired`, `on_hold`, `failed`
- Prisma schema: `paddleCustomerId` → `dodoCustomerId`, `paddleSubscriptionId` → `dodoSubscriptionId`
- Все legal pages обновлены (Privacy, Terms, Refund)

**Файлы:**
- `lib/dodo/client.ts` — Dodo SDK client (lazy init, build-safe)
- `components/billing/DodoCheckout.tsx` — компонент checkout кнопки
- `components/billing/AutoCheckout.tsx` — авто-checkout после регистрации (обновлён под Dodo)
- `app/api/billing/create-checkout/route.ts` — создание Dodo subscription + payment link
- `app/api/billing/webhook/route.ts` — обработка webhook событий с HMAC верификацией
- `prisma/migrations/20260324104500_rename_paddle_to_dodo/migration.sql`
